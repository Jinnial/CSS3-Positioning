Positioning Presentation

Sticky Example

The header of this section is a stick. As such, the element will be bound to the top edge of the screen until the end of the div it is contained it. It also only fixes itself to the edge when scrolled past.

.sticky {
position: -webkit-sticky; /* For Safari Only */
position: sticky;
top: 0;
}

NOTE: Bootstrap 4 allows for easier coding of positions. If using B4 all you have to do is add sticky-top as a class name.

Relative

The relative tag is used when you are positioning a page element relative to it's normal/original position. 

        
          #relativeCode {
            background-color: lightgrey;
            border-radius: 25px 25px 25px 25px;
            width: 450px;
            height:300px;
          }

          #code {
            position: relative;
            right:35px;
          }

This code in this snippet was moved 35px from the right by using the relative positioning.
